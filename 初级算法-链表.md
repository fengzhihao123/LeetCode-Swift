### 删除链表的倒数第N个节点

#### 说明
```
给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。
示例：
给定一个链表: 1->2->3->4->5, 和 n = 2.
当删除了倒数第二个节点后，链表变为 1->2->3->5.
```
#### 思路

* 快慢指针

#### 代码
```
func removeNthFromEnd(_ head: ListNode?, _ n: Int) -> ListNode? {
    var fastNode = head
    var slowNode = head

    for _ in 0..<n {
        fastNode = fastNode?.next
    }

    if fastNode == nil {
        return head?.next
    }

    while fastNode?.next != nil {
        fastNode = fastNode?.next
        slowNode = slowNode?.next
    }

    slowNode?.next = slowNode?.next?.next
    return head
}
```

### 反转链表

#### 说明
```
反转一个单链表。
示例:
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```

#### 思路

迭代或递归地反转链表

#### 代码

```
func reverseList(_ head: ListNode?) -> ListNode? {
    var tempHead = head
    var reverseHead: ListNode?
    while tempHead != nil {
        let next = tempHead?.next
        tempHead?.next = reverseHead
        reverseHead = tempHead

        tempHead = next
    }
    return reverseHead
```

### 合并两个有序链表
#### 说明
```
将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 
示例：
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```
#### 思路
while 
#### 代码
```
func mergeTwoLists(_ l1: ListNode?, _ l2: ListNode?) -> ListNode? {
    var tempL1 = l1
    var tempL2 = l2
    var newNode: ListNode? = ListNode(0)
    let tempNode = newNode
    var val = 0

    while tempL1 != nil || tempL2 != nil {
        if let val1 = tempL1?.val, let val2 = tempL2?.val {
            if val1 <= val2 {
                val = val1
                tempL1 = tempL1?.next
            } else {
                val = val2
                tempL2 = tempL2?.next
            }
        } else if tempL1 == nil {
            val = tempL2!.val
            tempL2 = tempL2?.next
        } else {
            val = tempL1!.val
            tempL1 = tempL1?.next
        }
        let node: ListNode? = ListNode(val)
        newNode?.next = node
        newNode = newNode?.next
    }
    return tempNode?.next
}
```
