### 爬楼梯

#### 说明
```
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。
```
#### 思路

* 斐波那契数列

#### 代码
```
func climbStairs(_ n: Int) -> Int {
    if n <= 0 { return 0 }
    if n == 1 { return 1 }
    var pre = 1
    var cur = 2
    for _ in 2..<n {
        let temp = cur
        cur = cur + pre
        pre = temp
    }
    return cur
}
```
